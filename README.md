# VPN Advanced - Arquitectura Indetectable
Arquitectura profesional para gestión de VPN con rotación de IP y ofuscación en Termux.

## Estructura del Proyecto
- **configs/**: Archivos de configuración de protocolos.
- **scripts/**: Lógica de automatización y seguridad.
- **logs/**: Registro de eventos del sistema.
- **lists/**: Control de acceso y rotación.
